# Quality Issues

Status: maintained
Last Reviewed: 2026-06-01
Review Cadence: 90 days

这里用于存放质量问题的活动记录与归档记录。

## 目录说明

- `active/`
  正在处理中的质量问题。当前没有 active 问题时，不要求保留空目录。
- `archive/`
  已关闭或已接受的质量问题。每条记录应包含关闭日期与验证方式。

具体字段与关闭标准见 `../QUALITY_ISSUE_STANDARD.md`。

## 记录要求

活动问题至少说明：

- 问题背景
- 影响范围
- 复现或识别方式
- 关闭条件
- 当前处理状态

归档问题至少说明：

- 处理结果
- 关闭条件
- 验证方式
- 关闭日期
